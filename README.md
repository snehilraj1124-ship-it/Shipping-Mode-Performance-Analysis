# 📊 Sales Performance Dashboard using Microsoft Excel

A comprehensive Business Analytics project developed using Microsoft Excel to analyze sales performance, customer behavior, profitability, and key business metrics through interactive dashboards and data visualization.

---

## 📌 Project Overview

This project demonstrates how Microsoft Excel can be used as a powerful Business Analytics tool to transform raw sales data into meaningful business insights. The dashboard provides an executive-level view of organizational performance through KPIs, Pivot Tables, Pivot Charts, and interactive filters.

---

## 🎯 Objectives

- Analyze overall sales performance
- Track business KPIs
- Monitor profit and revenue
- Identify sales trends
- Compare regional performance
- Evaluate customer behavior
- Support data-driven business decisions

---
